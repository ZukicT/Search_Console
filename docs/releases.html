What's next

Exploring
Not scheduled yet

Ideas I'm weighing for future releases. Nothing here is promised or in a fixed order. I build what helps my workflow and yours.

On the list

  • Live visitor pulse: can the app show who is on your site right now? Search Console data is delayed, so I'm looking at whether another source or a near-real-time estimate is possible. Not sure yet.
  • More native reports: bring indexing, links, and security checks into the app instead of opening Safari for every answer.
  • Compare periods: this week vs last week on one screen, without exporting to a spreadsheet first.
  • Smarter Overview digests: a short what-changed summary when you open a property.
  • Richer widgets: more metrics at a glance on your home screen and Lock Screen.

Which of these would help you most? Something missing? Send feedback below or use in-app feedback in Settings.

Version 2.0.0

Latest
May 2026

Improvements

  • Performance and reliability improvements across Overview, Search, and property details
  • Clearer in-app messages for errors, warnings, and successes
  • Improved bug report flow with device info and optional screenshot
  • Settings and UI polish, including optional micro animations
  • Reorganized Settings with clearer sections and semantic icon colors

Mac native development is paused while we focus on the iOS App Store release.

Version 1.0.5

Stable
February 2026

Bug Fixes

  • Fixed crash when switching result type to Discover or Google News
  • No-data alert now uses system pop-up; result type reverts to last working selection

New Features

  • Multi-language support: German, French, Japanese, Korean, Portuguese, Simplified and Traditional Chinese
  • Full UI localization: settings, about, charts, Core Web Vitals, notifications, achievements

Improvements

  • Chart: localized axis labels and compact metric cards for all languages
  • Removed Last 24 hours date range: chart now uses reliable daily data only (API does not provide hourly data)

If you selected Discover or Google News and saw no data, the app now shows a simple alert and switches the dropdown back to your last working view.
